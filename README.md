# Manix

A fast CLI documentation searcher for Nix.

## Usage

```sh
manix optionals
```

![manix](/manix.png)

## Installation

### Nix

```sh
nix-env -i -f https://github.com/mlvzk/manix/archive/master.tar.gz
```
