# Manix

A fast CLI documentation searcher for Nix.

## Usage

```sh
manix optionals
```

![manix](/manix.png)

## Installation

### Nix

```sh
git clone https://github.com/mlvzk/manix
cd manix
nix-env -f default.nix -iA manix
```
